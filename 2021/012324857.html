axios

网络请求 axios
axios
发布日期:   2021-01-23
阅读次数:  

一、 axios是什么？

对此，官方文档是这样说明的：

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。

二、 axios特性

  1. 可以在浏览器中发送XMLHttpRequests
  2. 可以再node.js中发送http请求
  3. 支持Promise API
  4. 拦截请求和响应
  5. 转换请求数据和响应数据
  6. 能够取消请求
  7. 自动转换JSON数据
  8. 客户端支持防御XSRF攻击

三、使用场景

在浏览器或Node.js环境中发送网络请求都可以使用axios库。

四、如何使用

安装模块
npm install axios

直接引入
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>

五、示例

  1. 发送一个GET请求

//不写方法默认使用get，一般还是指定方法比较好。
axios('https://www.xxx.com')
    .then(function(response){
        //数据处理...
    })
    .catch(function(error){
        //失败反馈信息...
    })

//指定get方法
axios.get('/user?id=123')
    .then(function(response){
        console.log(response); //发送请求成功后执行此语句
    })
    .catch(function(error){
        console.log(error); //失败执行此语句
    })

//以上请求也可以通过这种方式发送
axios.get('/user',{
    params:{
        id: 123
    }
})
.then(function(response){
    console.log(response)
})
.catch(function(error){
    console.log(error)
})
  1. 发送一个post请求
//post方法
axios.post('/user',{
    firstName: 'San',
    lastName: 'Zhang'
})
.then(function(response){
    console.log(response)
})
.catch(function(error){
    console.log(error)
})
  1. 一次性发送并发请求——all方法
//第一个请求方法
function getUserName(){
    return axios.get('/user/name');
}
//第二个请求方法
function getUserAge(){
    return axios.get('/user/age');
}

axios.all([getUserName(), getUserAge()])
    .then(axios.spread(function(result1,result2){
        //当两个请求都完成时会触发这个函数，两个参数分别代表返回的结果。
    }))

六、传递配置发送请求

除了以上的方式外，也可以通过向axios传递相关配置来创建请求。


// 发送一个post请求
axios({
    method: 'post',
    url: '/user/123',
    data: {
        firstName: 'San',
        lastName: 'Zhang'
    }
})

七、常见的配置选项

  • 请求地址
      url: '/user'
  • 请求类型
      method: 'get'
  • 请求根路径
      baseURL: 'http://www.xxx.com/api'
  • 请求前的数据处理
      transformRequest: [function(data){}]
  • 请求后的数据处理
      transformResponse: [function(data){}]
  • 自定义的请求头
      headers: {'x-Requested-With':'XMLHttpRequest'}
  • URL查询对象
      params: {id: 123}
  • 查询对象序列化函数
      paramsSerializer: function(params){}
  • request body
      data: {key:'aaa'}
  • 超时设置s
      timeout: 1000
  • 跨域是否带Token
      withCredentials: false
  • 自定义请求处理
      adapter: function(resolve, reject, config){}
  • 身份验证信息
      auth:{uname: 'admin', pwd:'123456'}
  • 响应的数据格式 json/blob/documentarraybuffer/text/stream
      responseType: 'json'

八、默认配置

可以设置默认配置，配置对所有请求都有效

  1. 全局默认配置。

axios.defaults.baseURL = 'http://xxx.com';
axios.defaults.timeout = 2000
  1. 自定义的实例默认配置（如何创建一个自定义实例的方法在下面会说明）。

//当创建实例的时候配置默认配置
var instance = axios.create({
    baseURL: 'https://api.example.com'
});

九、创建axios的实例

为什么要创建axios的实例呢？

  • 当我们从axios模块中导入对象时，使用的实例是默认的实例。
  • 当给该实例设置一些默认配置时，这些配置就被固定下来了。
  • 但是后续开发中，某些配置可能会不太一样。
  • 比如某些请求需要使用特定的baseURL或者timeout或者content-Type等。
  • 这个时候，我们就可以创建新的实例，并且传入属于该实例的配置信息。

创建axios实例示例

// 创建了一个axios实例
let instance = axios.create({
  baseURL: "https://xxx.com/api/",
  timeout: 2000,
});

//第一个请求
instance({
    url: '/home/multidata'
}).then(res => {
    console.log(res);
})

// 第二个请求
instance({
    url: '/home/data',
    params: {
        type: 'pop',
        page: 1
    }
}).then(res => {
    console.log(res)
})
